Exercise 1: Find the maximum

Exercise 2: Print the list, backwards

The existing printList function uses recursion to print the contents of the list. Write (and call) a printListBackwards function that will print the list backwards; that is, from the last item to the first item, using recursion.

Exercise 3: Power-of-Three

A number like 81 is an integer power of three (34) but 30 is not. There are a number of different ways of determining if a number is an integer power of three, such as by using logarithms or other mathematical tricks. But there is also a simple inelegant solution: keep dividing the number by 3, until you get a number that is not evenly divisible by 3 (then the original number is not a power of three) or until you reach 1 (then the original number is a power of three).

For this exercise, download the program Lab8Ex3.java and complete the isPowerOfThree function to use the inelegant solution to determine if n is a power of three, using recursion. Again, the function must be recursive, and use no loops.

Note:

If you want to implement the solution using iteration first, you may then find it easier to write a recursive version.
